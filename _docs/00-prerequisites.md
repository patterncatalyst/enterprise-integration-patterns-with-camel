This chapter will walk you through setting up your local development environment for the Enterprise Integration Patterns tutorial.

## What you will need

| Tool | Version | Purpose |
|------|---------|---------|
| Java | 21 (LTS) | Runtime for Quarkus and Camel |
| Maven | 3.9+ | Build tool |
| Podman | 5.x+ | Container runtime for the local stack |
| podman-compose | latest | Compose support for Podman |
| `gh` | latest | GitHub CLI |
| IDE | VS Code or IntelliJ | With Quarkus extension |
